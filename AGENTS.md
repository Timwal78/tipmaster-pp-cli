| `tipmaster status` | Service health and feature flags |

## Key Use Case for Agents

The `resolve` command is the primary agent use case — it lets any AI agent
look up an XRPL wallet address by Farcaster username without asking the user
for their wallet. This enables fully autonomous RLUSD tipping flows.

```bash
# Resolve wallet, then use Ghost Layer to bridge a tip
WALLET=$(tipmaster resolve dwr --compact | jq -r '.wallet_address')
ghost-layer bridge --chain XRPL --recipient "$WALLET" --amount 1000000 ...
```
